# Phase 1 — Theme Port: Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace PaperMod with mlaify's Tailwind + Pagefind theme as the foundation of matthewd.xyz, rendering the existing site content at parity, with a copper personal accent and Giscus comments preserved. Phases 2 (content migration) and 3 (redirects) follow as separate plans.

**Architecture:** Port mlaify's custom Hugo theme into this repo (no Hugo modules — files live directly in `layouts/`, `assets/`, `config/`). Build pipeline becomes `npm ci && hugo --gc --minify && pagefind --site public`. CSS uses Tailwind with a copper-accent palette (replacing mlaify's indigo brand) and warm-stone neutrals (replacing mlaify's slate ink). The CSS custom-property accent system (`--accent-*` overridable via `data-accent="..."`) is preserved so Phase 2's project pages get cyan (Aegis) and amber (AttackMap) for free. Giscus, currently wired in `layouts/partials/comments.html`, moves to `layouts/_partials/comments.html` and is mounted only on `/posts/*`.

**Tech Stack:** Hugo extended (0.153.2), Tailwind CSS 3, Pagefind 1.4, PostCSS, Autoprefixer, Inter + JetBrains Mono (Google Fonts), @tabler/icons (via Hugo module mount), Giscus (existing), Prettier.

**Scope boundary:** Phase 1 keeps `content/posts/` at its current location (no rename to `/writing/`) and does not import any mlaify content. Nav for Phase 1 is **Posts · About · Search**. Projects nav item and content are added in Phase 2.

**Source repo paths used in this plan:**
- This repo: `/Volumes/Dev/repos/GitHub/mdavistffhrtporg/mdavistffhrtporg.github.io`
- mlaify (read-only reference): `/Volumes/Dev/repos/GitHub/mlaify/mlaify.github.io`

When this plan says "copy from mlaify", it means copy from the second path.

- [ ] **Step 1: Replace `.github/workflows/hugo.yml` with this content**

```yaml
name: Build and deploy
on:
  push:
    branches:
      - main
  workflow_dispatch:
permissions:
  contents: read
  pages: write
  id-token: write
concurrency:
  group: pages
  cancel-in-progress: false
defaults:
  run:
    shell: bash
jobs:
  build:
    runs-on: ubuntu-latest
    env:
      DART_SASS_VERSION: 1.97.1
      GO_VERSION: 1.25.5
      HUGO_VERSION: 0.153.2
      NODE_VERSION: 24.12.0
      TZ: US/Chicago
    steps:
      - name: Checkout
        uses: actions/checkout@v5
        with:
          fetch-depth: 0

      - name: Setup Go
        uses: actions/setup-go@v5
        with:
          go-version: ${{ env.GO_VERSION }}
          cache: false

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: ${{ env.NODE_VERSION }}
          cache: "npm"

      - name: Setup Pages
        id: pages
        uses: actions/configure-pages@v5

      - name: Create local bin
        run: mkdir -p "${HOME}/.local"

      - name: Install Dart Sass
        run: |
          curl -sLJO "https://github.com/sass/dart-sass/releases/download/${DART_SASS_VERSION}/dart-sass-${DART_SASS_VERSION}-linux-x64.tar.gz"
          tar -C "${HOME}/.local" -xf "dart-sass-${DART_SASS_VERSION}-linux-x64.tar.gz"
          rm "dart-sass-${DART_SASS_VERSION}-linux-x64.tar.gz"
          echo "${HOME}/.local/dart-sass" >> "${GITHUB_PATH}"

      - name: Install Hugo
        run: |
          curl -sLJO "https://github.com/gohugoio/hugo/releases/download/v${HUGO_VERSION}/hugo_extended_${HUGO_VERSION}_linux-amd64.tar.gz"
          mkdir "${HOME}/.local/hugo"
          tar -C "${HOME}/.local/hugo" -xf "hugo_extended_${HUGO_VERSION}_linux-amd64.tar.gz"
          rm "hugo_extended_${HUGO_VERSION}_linux-amd64.tar.gz"
          echo "${HOME}/.local/hugo" >> "${GITHUB_PATH}"

      - name: Verify installations
        run: |
          echo "Dart Sass: $(sass --version)"
          echo "Go:        $(go version)"
          echo "Hugo:      $(hugo version)"
          echo "Node.js:   $(node --version)"
          echo "npm:       $(npm --version)"

      - name: Install Node dependencies
        run: npm ci

      - name: Configure Git
        run: git config core.quotepath false

      - name: Cache restore
        id: cache-restore
        uses: actions/cache/restore@v4
        with:
          path: ${{ runner.temp }}/hugo_cache
          key: hugo-${{ github.run_id }}
          restore-keys: hugo-

      - name: Build site
        run: |
          hugo \
            --gc \
            --minify \
            --baseURL "${{ steps.pages.outputs.base_url }}/" \
            --cacheDir "${{ runner.temp }}/hugo_cache"

      - name: Build Pagefind index
        run: npx pagefind --site public

      - name: Cache save
        uses: actions/cache/save@v4
        with:
          path: ${{ runner.temp }}/hugo_cache
          key: ${{ steps.cache-restore.outputs.cache-primary-key }}

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./public

  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    needs: build
    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```
